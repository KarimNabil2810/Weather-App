# Weather Forecast App

A modern weather application with glassmorphism design, built with React, Vite, and Material-UI. Features real-time weather, 5-day & 24-hour forecasts, voice search, favorites, city comparison, light/dark mode, AQI, and weather alerts.

**Tech Stack:** React 18, Vite, Material-UI, Axios, OpenWeatherMap API, CSS Modules

**Installation:** Clone repo → `npm install` → Add API key in `WeatherPage.jsx` → `npm run dev`

**Key Features:** City autocomplete, voice search, favorites, weather comparison, share/export data, print reports, light/dark mode, responsive design

**API:** OpenWeatherMap (current weather, 5-day forecast, geocoding, air pollution)
